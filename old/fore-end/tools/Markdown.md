## Markdown

### 强调
```
*斜体*	**粗体**
```

### 链接
```
[描述](http://example.com)
```

### 图片
```
[描述](http://example.com)
```

### 笔记本
```
@(笔记本)[标签1,标签2,标签3]
```

### 标题
```
标题1		标题2
========	--------

## 标题2		###### 标题6
```

### 列表
```
1. 有序列表	- 无序列表      - [ ] 复选框
2. 有序列表	- 无序列表      - [x] 复选框
```

### 引用
```
> 这是引用的文字
> 引用内可以嵌套标题、列表等
```

### 代码
```
这是一句行内代码 `var a=1` , 以下是代码区块：
	```ruby
	print 'Hello world'
	```
```

### LaTex 公式
```
这是一句行内公式 $ y = x + 1 $ , 以下是整行公式：
$$ a^2 + b^2 = c^2 $$
```

### 表格
```
| Item      |    Value | Qty  |
| :-------- | --------:| :--: |
| Computer  | 1600 USD |  5   |
```
